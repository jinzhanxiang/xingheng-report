二、纵向分析——时间轴上的竞争力演变

2.1 第一阶段：创立与生存期（2003-2010）

2.1.1 创立背景

星恒电源成立于2003年，由中科院物理所陈立泉院士团队技术孵化、联想控股早期投资。第一个产品是锰酸锂圆柱电池（21115型号），用于电动自行车铅酸电池替代。

创立时的战略逻辑：

  • 选择巨头（宁德时代尚未成立，比亚迪聚焦汽车）看不上的边缘市场

  • 以锰酸锂技术路线起步——成本低（无钴无镍）、工艺成熟（基于中科院物理所积累）

  • 目标：在"小池塘里做大鱼"，等待两轮车锂电化浪潮

来源：[C] 百度百科/腾讯新闻/星恒官网公开资料；[B] 项目材料

3.1.6 深度补充：市场出清机制与防御性竞争力的再审视

前述分析将市占率提升100%归因于分母收缩（Shapley分解：市场收缩贡献126%，出货下降抵消26%），结论是"防御性强于进攻性"。但这一结论回避了一个更深层的问题：如果市场收缩45.8%而星恒出货仅降11.6%，这本身就说明星恒的抗跌能力显著强于市场均值。以下从市场出清的细分结构、幸存者特征、防御性竞争力的边界条件三个维度，对这一悖论进行系统分析。

  • 成本优势是最关键的差异化因素——在GM仅13.6%的微利行业，10-15%的成本优势就是"生与死"的差别

  • 共享转型是窗口期抓得最准的战略决策——在OEM渠道萎缩前成功开辟了第二战场

  • 专注度既是主动选择也是被动结果——业务高度集中使星恒成为"最后一个没有退路的人"

  • 渠道锁定提供了惯性保护——但在行业复苏时可能转化为劣势（新客户仍不愿被锁定）

这一分析对估值的关键含义是：星恒的市占率72.5%虽然主要源自市场收缩（Shapley分解：行业贡献126%），但其在收缩中的存活能力（出货仅降11.6% vs 市场降45.8%）确实是真实竞争力的体现。这与"市占率完全来自幸存者偏差、毫无竞争力含量"的极端负面解读不同。

I. 合规口径 vs 全口径——市场结构的深层解读（点击展开，2026-06-10 新增）

(2) 为什么星恒在非合规市场参与较少？——三重壁垒分析

第一重：认证与合规壁垒（主动选择不参与）

星恒作为行业头部品牌，是GB 43854-2024锂电池安全白名单的首批入选企业（全国仅9家）[D]。这意味着：

  • 星恒的每款产品都通过了CCC认证——认证成本单型号数十万元，周期6-12个月

  • 合规成本（认证费+检测费+质量控制+售后保障）在星恒的成本结构中占比约3-5%

  • 白牌作坊完全不承担这些成本——这是白牌能卖到品牌电池40-60%价格的核心原因

  • 星恒若参与白牌市场，将面临：(a) 合规认证被吊销的风险，(b) 安全事故的品牌连带责任，(c) 合规产品与自有白牌产品的自我蚕食

三层出清在白牌市场中的叠加效应：

第一层（成本出清，2021-2022）：碳酸锂暴涨50万/吨
  → B品电芯价格同步上涨 → 白牌成本优势缩窄
  → 白牌电池与品牌电池价差从50%缩小至20-30%
  → 部分白牌组装商因利润归零而退出

第二层（价格战出清，2022-2025）：合规品牌降价
→ 星恒/ATL/天能等品牌电池降价 → 进一步压缩白牌价差空间
→ 品牌电池ASP降至400-600元/组 → 与白牌价差缩小至100-200元
→ 消费者愿意为100-200元的价差选择有安全保障的品牌产品


第三层（合规出清，2024-2025）：监管执法+平台治理
→ GB 43854-2024正式实施（2025年11月1日起禁售未认证电池）
→ 拼多多/淘宝2024年起清理无CCC认证的锂电池listing
→ 各地市场监管部门2024-2025年开展电动车安全隐患整治
→ 售后改装渠道因消防检查和处罚风险而被封堵

E. 白牌市场清零的估值含义

白牌市场从~386万组→趋近于0，对星恒估值的含义是双向的：

正面：

  • 星恒的合规市场市占率从~80%（2021年白牌存在时）降至72.5%（2025年白牌清零后），但可竞争市场的"质量"显著提高——合规市场的进入壁垒（CCC认证等）远高于之前的全口径市场

  • 白牌清零后释放的市场需求部分转化为合规品牌产品的增量——星恒作为合规龙头是最大受益者

  • 合规市场~474万组虽比全口径市场小，但利润率更高（白牌将行业ASP拉低了约20-30%）

负面：

  • 星恒的72.5%市占率描述的是一个正在收窄的市场——白牌清零意味着行业总可竞争市场从~874万组收缩至~474万组

  • 如果GGII的30%+口径正确（~1,144万组的全口径），那星恒的非合规市场"缺席"意味着失去了约670万组的潜在市场——无论是因为"不能参与"还是"不想参与"

  • 在合规市场中，星恒面临的是更强大的竞争对手（ATL新能安、比亚迪等），而非脆弱的白牌组装商

F. 集成判断（置信度：★★★★☆）

星恒选择不参与非合规市场是理性的战略选择而非能力不足——其成本结构（固相法）虽然优于合规竞争者，但无法与不承担安全成本的白牌竞争。这一选择的结果是：

  1. 星恒在合规市场的份额（72.5%）远高于其在全口径市场的份额（GGII 30%+）

  2. 合规市场虽然体量更小（474万组 vs 全口径1,144-1,900万组），但进入壁垒更高、利润率更好、增长更可持续

  3. 白牌市场的政策性清零（GB 43854-2024）正在将两个口径合二为一——2025年的474万组合规市场就是未来的全口径市场基线

  4. 天保控股收购的是合规PACK龙头而不是全口径龙头——这一定位的投资含义需要在估值中体现

E. 关键结论

  1. "下游一定比上游赚钱"被多行业实证数据证伪。决定因素不是"上下游位置"，而是技术壁垒、集中度对比、产品差异化、下游品牌壁垒四个条件的组合。

  2. 星恒在两轮车锂电中的合理净利率中枢约2.5-4.0%，保守估计不低于2.0%。天能动力（铅酸为主，技术壁垒更低）做到3.09%，星恒作为锂电专业化企业应有不低于此的水平。

  3. FY2025的1.24%低于星恒合理稳态水平——这是储能亏损（-0.27亿）+价格战底部+补贴退坡三重压力下的阶段性低估。随着储能改善和行业价格回升，净利率向2-3%回归的概率较高。

  4. FY2026管理层1.5亿（4.1%净利率）：处于中位偏上区间，并非离谱但需要储能扭亏+海外扩张+ATL竞争趋缓三者同步发生。概率加权均值0.8-1.1亿（2.2-3.0%）低于管理层目标。

  5. FY2020的9.5%净利率不可作为估值基准——该峰值依赖的汽车电芯锁价业务（价格1.32-1.386元/Wh vs 当前0.48元/Wh）已结构性消失。

  6. 对估值的影响：DCF模型中星恒稳态净利率应设为2.0-3.0%（以天能3.09%为锚，下调0.5-1pp以反映ATL竞争压力），而不应沿用FY2020的9.5%或FY2025的1.24%作为永续假设。

4.3 四个关键战略选择的长期后果

选择一（2003）：边缘市场根据地

当时：选择巨头看不上的两轮车电池市场 结果：20年后成为该细分市场全球第一（72.5%） 代价：天花板低（FY2025纯锂电市场规模约40-50亿），当巨头开始"下沉"时规模劣势暴露 长期判断：这是一个明智的创业选择，但也是一个有明确天花板的赛道选择。星恒没有在2015-2020动力电池爆发窗口完成"出圈"——详见选择二

选择二（2015-2020）：留在两轮车不"出圈"

当时：动力电池行业爆发（宁德时代/比亚迪/亿纬锂能快速崛起），星恒选择继续聚焦两轮车 结果：避免了与巨头的正面竞争，但也错过了中国新能源产业最大的成长窗口 反事实分析（置信度：★★★☆☆）：

  • 支持"能力约束"的论据：2015年的星恒（营收约10-15亿级）进入动力电池面临资本壁垒（单GWh投资3-4亿 vs 两轮车0.5-1亿）、客户认证壁垒（汽车OEM 2-3年 vs 两轮车6-12月）、技术壁垒（锰酸锂不是汽车主流路线）

  • 支持"主动选择"的论据：两轮车锂电化是新国标驱动的确定性赛道，深耕优势领域是理性选择

7.1 竞争力趋势

  1. 星恒的两轮车锂电龙头地位来自20年专注+对手退出，而非不可复制的竞争壁垒。ATL新能安在7年内从0做到18.50%，证明该市场不存在不可逾越的进入壁垒。

  2. 市占率72.5%不宜被解读为"垄断性竞争优势"——其100%的"提升"来自分母收缩（市场总量从874万→474万组）——Shapley分解：市场收缩贡献126%，星恒自身出货下降抵消-26%。在绝对出货量上，星恒2025年出货343.34万组仍低于2020年峰值481.71万组（-28.7%）。

  3. 竞争格局已从"一超多强"演变为"双寡头"——星恒72.5% vs ATL 18.5%，其余不足5%。但双寡头格局并不稳定——ATL是进攻方，星恒是防守方。

  4. 财务竞争力自FY2020以来单边下滑——以接近历史峰值的营收（36.36亿 vs 40.21亿）仅创造1/8的净利（0.45亿 vs 3.76亿）。

  5. 标准参与构成隐形竞争壁垒——此前未被纳入分析（2026-06-03新增）：星恒是GB/T 36972-2018（电动自行车用锂离子蓄电池，国内该领域最核心的技术标准）的第一起草单位，并正在主编国家强制性标准（推断为GB 43854-2024配套标准）。在同业中，ATL/博力威/天能均未在核心标准中担任第一起草单位。标准参与从三方面强化竞争力：(a) 先发优势——提前知晓标准指标方向；(b) 准入门槛——若标准参数基于星恒产品数据设定，竞品需额外研发投入；(c) 客户信任——标准主编身份在OEM采购评估中为加分项 [B/总-星恒-标准.xlsx, 置信度B级]。但注意：标准参与不等于产品实际合规（见04报告2.5节——星恒在过充电保护测试中被多次抽检不合格），其竞争壁垒效应受产品执行质量的约束。

7.2 核心结论

星恒的竞争力处于**"表面稳固、实质脆弱"**状态。72.5%的市占率（2025年）是行业洗牌的"幸存者红利"而非竞争制胜的结果。当竞争对手退出基本出清（2025年其他合计仅4.49%），星恒将直接面对ATL新能安——一个拥有CATL成本协同+ATL品质体系的竞争对手——的单挑。届时星恒的市占率将面临"回归均值"的压力。固相法磷酸锰铁锂提供了约3年（2026-2029）的差异化窗口，但窗口期结束后如果没有重大突破，星恒将回归为"锰系两轮车电池制造商"——一个市场规模有限（FY2025纯锂电约40-50亿）、利润率微薄（FY2025净利率1.24%、长期稳态估计1-2%）、技术天花板清晰的赛道。

一、分析框架：为什么行业vs企业归因是投资的底层逻辑

1.1 两类因素对估值的影响机制

行业原因（外生）
影响范围: 影响所有行业参与者
估值调整: 调整可比公司PS/PE倍数（行业β）
关键判断: 行业周期方向+持续时间
案例: 锂盐价格下跌 → 全行业电芯厂毛利率承压 → 行业PS中枢下移
企业原因（内生）
影响范围: 仅影响星恒特定经营
估值调整: 在公司PS倍数基础上叠加额外折价（公司α）
关键判断: 该因素是否可修正？修正需要多长时间？
案例: 启源纳川债务危机 → 星恒股权冻结 → 公司特定治理折价
✅ 提示

投资决策含义：如果一个问题被判定为行业原因，那么天保收购星恒时不应仅因这一因素而对星恒施加过大的公司特定折价——因为竞争对手面临同样的问题。如果是企业原因，则需要评估该问题是否会在天保入主后得到解决——如果可以解决，折价应反映的是"解决问题的时间和成本"，而非永续折价。

四、企业原因详细分析——根因追溯与可修正性

4.1 合同与客户结构——星恒最深层的内生问题

4.1.1 定价权缺失：合同条款的单向不对称

为什么是星恒的原因而非行业原因？

对比验证（同业可比口径）：博力威同样供应两轮车OEM（九号电动为主），但直接对比须严格区分口径：(1) 博力威FY2025整体GM 18.08% [A] 含海外高GM储能业务（营收27.35亿，+48.3%主要来自海外储能拉动 [附件D第3.1.3节]）；(2) 博力威未单独披露国内两轮车锂电段GM，行业合理估计为5-10%区间（与星恒国内民用6.6%相近）；(3) 因此，博力威整体GM 18.08% ≠ 国内两轮车OEM段GM，不可直接用于论证星恒定价权问题为"企业特有"。同口径下（国内两轮车锂电），博力威与星恒的毛利率差异远小于整体GM的18.08% vs. 6.6%的表观差距。

  • 中期（3-5年）：若ATL主攻高端、星恒稳住中低端，两方的客户群差异化→各自在自己的细分市场拥有定价权

4.2.3 华霆锁价合同到期降价↓64%

为什么是星恒的原因？

对比验证：锁价合同是商业谈判的结果——不同供应商面对同一客户的定价策略不同。ATL进入两轮车市场时并未采用类似的长期锁价策略。星恒与华霆签订从1.32-1.386元/Wh（2020年）暴跌至0.48元/Wh（2021年）的定价条款，是特定的商业决策失误——定价缺乏下行保护机制。

根因分析：

  • 星恒在2019-2020年为获取华霆汽车电芯订单，接受了高频定价协议（每月/每两月签一次价格协议）

  • 当市场需求变化时，华霆可以逐月压低采购价格——星恒无长期锁价保护

  • 这反映了星恒在汽车电芯领域的弱势谈判地位——与两轮车OEM类似的问题

修正路径：

  • 华霆已在FY2025占比较小（客户结构已变化）

  • 未来签新客户时应设置定价下限或价格调整带宽

  • 该问题对当前估值影响有限——是历史问题而非未来问题

4.3 技术与产品——企业战略选择的结果

4.3.1 锰酸锂技术路线的战略选择

为什么既有行业原因又有企业原因？

行业原因（30%）：2003年时锰酸锂确实是两轮车电池的合理选择（成本低、安全性好、无钴无镍）。磷酸铁锂当时尚未成熟（比亚迪刀片电池2020年才推出），镍钴锰三元成本过高。星恒选择锰酸锂在当时的技术条件下是合理的。

企业原因（70%）：但星恒持续固守锰酸锂路线，而ATL用磷酸铁锂/镍钴锰三元进入并迅速抢占高端市场——说明锰酸锂在高端市场的竞争力存在天花板。星恒的磷酸锰铁锂转向（2019-2022年启动）偏晚——如果2015-2016年就开始布局，现在的时间窗口会更宽裕。

修正路径：

  • 磷酸锰铁锂固相法是星恒修正锰酸锂天花板的关键赌注

  • 窗口期约3年（2026-2029），需在此期间实现万吨级产能+标杆客户突破

  • 如果固相法磷酸锰铁锂成功确立成本优势，锰酸锂路线选择的"历史包袱"将转化为"差异化壁垒"

4.3.2 储能起步晚、规模小（0.64GWh）

为什么是星恒的原因？

对比验证：宁德时代储能出货量超70GWh（2025年），阳光电源储能系统出货量超20GWh。星恒0.64GWh的储能规模反映了其进入储能市场的时间晚、投入力度不足。虽然"70%行业价格战+30%企业规模劣势"的拆分说明行业因素是主导，但储能起步晚是企业战略选择的结果——星恒的资源和注意力长期集中在两轮车锂电，储能被视为"副业"。

修正路径：

  • 2-3年内储能GM改善至0-3%是合理预期；改善至10%+则需要行业价格战缓解+客户结构优化+规模效应显现
