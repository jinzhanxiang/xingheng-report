1.1.5 净利率归因：毛利率差距 vs 费用率差距

星恒净利率低于博力威，具体多少归因于毛利率差距、多少归因于费用率差距？以FY2025数据做精确归因：

对比维度 博力威（FY2025） 星恒（FY2025） 差异

四项费用率 14.67% 13.15% +1.52pp（星恒更低）
费用后经营利润率 3.41% 0.48% -2.93pp

归因分解：

  • 毛利率差距贡献：-4.45pp → 这是净利率差距的全部来源，甚至更多

  • 费用率优势抵消：+1.52pp → 星恒用更低的费用率弥补了一部分毛利率劣势

  • 净效应（含其他收支差异）：-0.94pp

结论：星恒净利率低于博力威，100%归因于毛利率差距。如果星恒维持现有费用率（13.15%）但毛利率达到博力威水平（18.08%），其净利率可达约5%，ROE可达~7%。

1.1.7 波动性根因分析：为什么星恒的毛利率波动是行业最高的？

星恒毛利率标准差5.58，而博力威仅1.12。这一差异不是偶然的，可以从三个结构脆弱性解释：

脆弱性一：客户集中度——单一大客户的年度议价直接冲击毛利率。 哈啰出行占星恒毛利约23%（商调报告附件G），共享运营商年度集采谈判中，每1pp的价格让步直接对应约1pp的毛利率损失。博力威的客户结构更为分散（前五大客户集中度较低），单一客户的冲击有限。

脆弱性二：储能业务盈亏不稳——大幅波动且持续亏损。 星恒储能及新业务FY2022毛利率-32.09%、FY2024改善至-1.77%、FY2025回落至-5.51%。储能业务盈亏状态波动对综合毛利率的贡献幅度达数个百分点。

脆弱性三：固定成本刚性——营收波动直接放大为毛利率波动。 星恒FY2023营收下降33.75%，从35.99亿骤降至23.84亿。固定资产净值约20.95亿（[B] Q1 2026合并报表），年折旧约1.2-1.5亿（估算——附件D line 52明确标注「未获取固定资产折旧明细表进行精确验证」，置信度⭐⭐⭐），在营收萎缩时固定成本无法同步压缩，导致毛利率从9.79%崩塌至2.44%。而亿纬锂能FY2023营收反而增长34%（363→488亿），规模扩张摊薄了固定成本。

1.5.5.5 海外市场对估值的折溢价影响——方法论检验

推论⭐⭐⭐⭐
三位一体检验——海外市场应否作为独立折溢价因子？

检验维度一：海外高利润率是否已隐含在财务指标中？
→ 是。海外22.3%毛利率已计入FY2025合并毛利率13.63%和净利率1.24%。
在上述可比公司PS的三种锚定方法（利润率调整/去铅酸化/两轮车可比）中，
财务指标是PS推导的直接输入——海外利润贡献已系统性反映在PS锚定中。


检验维度二：海外定位是否独有于可比公司？
→ 否。博力威出口比例（~95%）远超星恒（17.3%），海外是其核心增长引擎。
ATL新能安全球电摩锂电市占率第一。海外高利润是行业共性方向，
星恒在海外定位上并不领先于可比公司。单独给予海外溢价缺乏行业基准支撑。


检验维度三：东南亚期权价值是否足够大且确定，足以支撑独立溢价？
→ 否。三项子检验：(a) FY2025东南亚仅2.20亿（6.1%营收），占比较小；
(b) 油改电政策落地节奏不确定（印尼目标vs实际差距大）；
(c) Amara Raja客户关系流失证明海外客户不确定性高。
期权价值具有"规模太小、不确定性太大"的特征。


综合结论：海外市场不应作为独立溢价或折价因子单独调整。
理由：(1)海外利润已通过财务指标隐含在可比公司PS中；
(2)海外定位不独有于可比公司（博力威更出口导向）；
(3)东南亚增长期权不确定性抵消其潜在价值；
(4)管理层海外陈述可信度偏低削弱"故事溢价"；
(5)现有「市占率龙头溢价」（+5%至+10%）已部分覆盖海外增长潜力。

  但不构成独立估值溢价因子。核心原因：
  
  1. 不是独有优势——博力威出口比例（~95%）远超星恒（17.3%）
  2. 海外利润已隐含在可比公司PS定价中（财务指标是PS的直接输入）
  3. 东南亚油改电期权存在但规模太小（2.2亿）+不确定性太大
  4. 资源约束明确（核心产线满产+可用资本有限）

1.5.5.7 国内竞争优势在东南亚的可迁移性分析

星恒在国内两轮车锂电市场建立了四层复合护城河（详见1.2节护城河溯源）。但东南亚两轮车市场与国内存在结构性差异，导致这四层优势在东南亚面临不同程度的迁移障碍。

第一层：LMO/锰酸锂材料体系——气候优势丧失

星恒的锰酸锂（LMO）技术路线在国内的核心优势是低温性能优异（-20°C容量保持率>80%），适配中国北方寒冷气候。但东南亚全年高温（25-35°C），LMO的低温优势在热带完全无法体现。相反，LMO在高温下的锰溶解问题是已知技术缺陷——东南亚高温高湿环境可能加速容量衰减。磷酸铁锂（LFP）在高温稳定性和循环寿命上优于LMO，这也是为何ATL新能安（LFP路线）在东南亚高端市场占据领先地位的技术原因。

气候适应性对比：
中国：低温是痛点 → LMO低温优势 = 差异化竞争力
东南亚：高温是痛点 → LMO高温劣势 → LFP更有竞争力
结论：LMO国内最大技术优势在东南亚不成立，反而可能成为劣势

第二层：32650大圆柱产品架构——BMS成本优势在铅酸主导市场无效

32650大圆柱电池通过减少电芯数量降低BMS复杂度，在国内市场可节省7-8pp的BMS成本。但东南亚两轮车市场仍以铅酸电池为主（燃油摩托车销量占比96.8%，电动化处于极早期，来源：Statista数据，博力威年报引述），锂电化处于起步阶段。在这个阶段：

  • 价格敏感度极高：消费者对锂电池的初始购置成本（vs铅酸）远比BMS成本结构敏感

  • 锂电池零售价约0.8-1.2元/Wh，铅酸电池约0.37-0.45元/Wh（来源：东海证券2026策略报告 + 天能股份债券募集书）；同容量下锂电池初始购置成本约为铅酸的2-3倍——BMS节省的7-8pp不足以弥合此价差

  • 铅酸电池回收残值约120元/组（vs锂电约40元/组），在东南亚以旧换新文化中进一步强化铅酸的经济性（来源：每日经济新闻，2025-07-13）

第三层：20年制造工艺积累——工艺经验无法跨市场移植

星恒在电解液配方、正极包覆等方面的工艺积累是针对LMO/LMFP体系的know-how。这些工艺优势体现在产品层面（循环寿命、安全性、一致性），但：

  • 产品层面的优势需要通过本地化客户验证和认证才能转化为市场优势

  • 东南亚客户（整车厂+经销商+终端消费者）对星恒品牌认知度极低

  • ATL新能安（全球电摩锂电市占率第一）在东南亚的渠道和认证布局远早于星恒；松下/LG/三星SDI在东南亚两轮车锂电领域无可见存在感——星恒的主要竞争对手是同样来自中国的ATL新能安和博力威

  • Amara Raja转向国轩高科证明：即使产品质量达到国际OEM准入标准，客户关系仍可能因竞争对手的战略性绑定而流失

五、产业链延伸：如果收购星恒，如何实现1+1>2？

ℹ️ INFO

ℹ️ 本章回答「如果收购星恒，应向哪个方向进行产业链延伸以实现协同效应」。这一分析建立在前面四章的结论之上：星恒的困境是结构性的（产业链位置劣势），单纯改善经营只能小修小补。真正的价值创造需要产业链位置跃迁。

5.1 为什么需要产业链延伸？

回顾星恒的核心问题：

推论⭐⭐⭐⭐
星恒的利润被两端挤压:
  
  上游（材料供应商）          ← 拿走35-40%的电芯成本（正极材料）
  星恒自身                   ← 毛利率13-17%区间波动，净利率仅1-2%
  下游（共享运营商/OEM）      ← 哈啰单一客户占毛利~23%
  
  星恒处于典型的「夹心层」位置:
  对上游议价权弱（碳酸锂为全球定价大宗商品，星恒是价格接受者；
    正极材料供应商分散CR3<30%但行业普遍亏损，降价空间有限）
  对下游议价权弱（共享运营商比主机厂更强势，合同无最低采购量保障）

如果通过并购向某一端延伸，星恒可以获得：

  • 向上游延伸：锁定原材料成本，捕获正极/前驱体环节的利润

  • 向下游延伸：锁定客户需求，绕过运营商直接触达终端用户

  • 横向延伸：进入相邻市场（储能/动力电池），扩大TAM

推荐度: ⭐⭐⭐。向上游前驱体延伸是自然方向但天花板低；正极材料并购是「便宜但可能更糟」（行业多数亏损）。

8.1 核心风险

  1. 碳酸锂价格超预期下跌（P0）: 当前16.5万/吨（数据来源：上海钢联MMLC电池级碳酸锂中间价165,100元/吨，2026-06-10）高于长期均衡价（8-12万）。若2026H2-2027年全球新增约26万吨LCE产能集中释放（基于各公司产能规划公告汇总，百川盈孚统计），锂价可能快速回落至10万以下。届时华友钴业/璞泰来PE将升至50x附近（粗略估算）。应对: 以盐湖股份+宁德时代为核心持仓
