附件F：天保控股收购星恒电源——收购方案与谈判议案

一、收购方案的演变与当前思路

1.1 原始方案及其分析

天保控股对星恒电源的收购方案根据实际情况不断改进和调整。最初形成了三个递进方案：

方案一：控股并表方案（13.73% + 启源纳川17.73%）

收购A组10家PE持有的13.73%老股，同时承接启源纳川持有的17.73%，合计持股约31.46%，实现控股并表。

该方案经分析后认为不可行，核心原因在于启源纳川的股权状态：纳川股份被执行22.33亿，启源纳川持有的星恒股权存在20条股权冻结及轮候冻结，无法直接交易转让。启源纳川的清理需要先解决其自身的债务问题，而这是一个复杂的司法程序，无法在短期内完成。因此，方案一在起点上就面临无法逾越的法律障碍。

方案二：13.73% + 产能落地

在第一阶段仅完成A组13.73%收购（不触及启源纳川），同步推进在天津落地10GWh产能——以产业投资换取地方政府支持，同时以产能合作深化与星恒的关系，为后续增持创造条件。

方案三：仅产能落地（JV合资）

不收购老股，天保与星恒在天津合资建设产能。这是最低风险的路径，但控制力最弱——天保仅作为合资方之一，对星恒主体无股权影响力。

方案一和方案二的核心依赖：均依赖收购老股（13.73%或更多）作为切入点。方案二在此基础上叠加了产能落地的对价。

1.2 现实变化与当前判断

变化一：工信部产能审批政策的不确定性

根据行业初步信息，工信部正在推进电池产能审批机制改革：审批权限上收至中央，实行指标制+分类管控，新建产能硬性门槛大幅抬高，新规真正收紧窗口从2027年下半年开始。上述信息尚未经工信部正式确认。

这一变化直接冲击了方案二（产能落地）和方案三（JV合资）中「新建产能」的可行性基础。委托加工模式（利用星恒现有产能，不涉及新建）成为实现天津产值落地的替代思路，但这仅是过渡性安排（详见第五章）。

建议在下周二会议中与星恒管理层商议，由星恒赴工信部确认具体政策内容和时间表。不排除产能置换等替代路径。

变化二：13.73%不足以实现天保的战略诉求

仅收购13.73%少数股权（约5亿），无法满足天保控股改善收入结构的核心诉求：

  • 13.73%为少数股东权益，无法并表——星恒约36亿营收无法计入天保体系合并报表

  • 对天保控股合并收入结构的改善效果为零

  • 少量股权的战略影响力有限——无法主导星恒的经营决策和产能分配

变化三：港股上市的估值不确定性

星恒计划以港股为上市目的地。当前港股市场对小市值制造业公司的估值环境存在显著不确定性——尤其是对于星恒这样分红不突出、处于竞争性行业的中等规模企业，市场给予的估值倍数可能低于预期。两三年后的港股市场环境更难以预测。这意味着即使IPO成功，退出估值也可能不理想，存在较大可能的本金损失风险。（准确的退出价值分析需参见独立的投资回报测算报告。）

当前倾向：基于上述变化，天保控股倾向于整体收购并表，并后续重新规划星恒的资本路线图（A股注入替代港股IPO）。本报告即为与星恒进行新一轮商讨之前的思路和方案框架。

重要提示：工信部产能政策信息为行业初步判断，尚未经工信部正式确认。

3.1 方案总览

4.2 装入天保基建的三种结构模式

星恒装入天保基建存在三种结构选择，核心问题在于星恒的进入与天保基建现有业务（地产、大飞机）的关系：

模式A：增量叠加——「地产+大飞机+星恒新能源」三主业

操作方式：天保基建以发股+现金收购星恒控股权（30%+/51%+/100%），不剥离现有任何业务板块。

优势：

  • 实施路径最简单——不需要处置现有资产，仅增加新业务板块

  • 不涉及地产剥离的复杂税务/债务/人员安排

  • 大飞机板块保留，维持估值锚（A股稀缺标的）

劣势：

  • 三主业结构可能导致资本市场给予「多元化折价」

  • 地产标签不消除——CSRC对地产上市公司跨界并购的窗口指导仍是障碍

  • 投资者叙事——「地产+大飞机+两轮车锂电」缺乏连贯的投资逻辑

可行性评估：中高。技术上可操作，但CSRC审核层面需论证三个业务板块的协同逻辑。预计审核周期12-18个月（预估）。

模式B：置换地产——「大飞机+星恒新能源」双主业（推荐）

操作方式：天保基建剥离房地产开发业务，同时以发股+现金收购星恒控股权，实现主业从「地产+大飞机」向「大飞机+新能源」的转型。

优势：

  • 消除房地产标签——摆脱地产标签后，CSRC跨界审核阻力大幅降低

  • 清晰的投资者叙事——「大飞机（高端制造）+星恒（新能源）」的双主线逻辑自洽，符合国家产业政策方向

  • 估值重估潜力——地产板块PB 0.67x拖累整体估值，剥离后剩余业务可能获得估值修复

  • 释放CSRC审核空间——非地产上市公司收购新能源资产，审核标准为正常的主业协同论证

劣势：

  • 地产板块剥离操作复杂——涉及项目公司、土地增值税、存量债务、人员安置

  • 天保基建当前营收高度依赖地产（具体占比需查阅2025年报），剥离后短期营收断崖

  • 地产剥离本身可能触发重大资产重组审核

地产板块剥离可行性：天保基建的地产板块以天津保税区园区开发为主，资产边界相对清晰（相对于全国性房企）。天保控股作为51.45%大股东，可以直接承接地产板块（关联交易）。园区开发类资产有一定的现金流，并非纯负资产。

模式C：置换大飞机——「地产+星恒新能源」（不推荐）

操作方式：天保基建剥离大飞机（中天航空）业务，以星恒替代。

不推荐的理由：

  1. 大飞机是估值锚——中天航空是A股稀缺的大飞机产业链标的，剥离等于放弃最具估值价值的资产

  2. 增长前景对比——大飞机板块（C919量产爬坡期）的增长前景优于地产

  3. 置换逻辑不成立——地产+星恒的组合并未解决「地产标签」的核心问题

结论：模式C仅作为理论选项列出，不具有实际推荐价值。

4.3 模式B（置换地产）的两种实施路径

如果采用模式B（置换地产+装入星恒），存在两种实施路径：

路径B1：天保控股先收购星恒 → 同步置入置出

操作步骤：

  • 第一步：天保控股以现金收购星恒控股权（30%+或51%+或100%）

  • 第二步（同步进行）：天保基建向天保控股发行股份+支付现金，收购天保控股持有的星恒股权（置入）；天保控股以现金收购天保基建的地产板块（置出）

优势：

  • 两步走结构清晰，每一步的审批边界明确

  • 置入置出同步操作，可在一次重组中完成主业转型

  • 天保控股在中间承担资产中转站角色，隔离了直接交易的操作风险

劣势：

  • 天保控股需在第一步独立出资收购星恒——资金压力集中（路径一5亿+第二步增持至30%+额外约3-6亿→路径二30%总计约8-11亿；若收购51%+则需约15亿+）

  • 天保控股同时承接地产板块需要额外资金（天保基建地产净资产规模待确认）

  • 两步之间的时间衔接风险——若第一步完成但第二步受阻（CSRC审核延迟/否决），天保控股同时持有星恒和面临资金压力

对天保控股的资金要求（路径B1，以收购星恒30%为例）：

资金用途 金额（预估） 来源
首期A组13.73% 约5亿 天保控股自有资金+银行并购贷款
第二步增持B组至30%+ 约3-6亿 银行并购贷款
承接天保基建地产板块 待确认净资产规模 天保控股自有资金/资产置换
合计（不含地产承接） 约8-11亿 —

路径B2：天保控股先置出地产业务 → 天保基建用现金+发股并购星恒

操作步骤：

  • 第一步：天保控股以现金收购天保基建的地产板块（置出）

  • 第二步：天保基建用置出回笼的现金+定向增发新股，直接并购星恒

优势：

  • 天保基建自己持有现金+发股工具，可直接并购星恒，不需要天保控股做中转站

  • 置出地产后，天保基建成为「大飞机+现金」平台，并购星恒的CSRC审核标准为正常产业并购

  • 天保控股的资金压力集中在承接地产板块，而非同时承担星恒收购

劣势：

  • 两步之间有先后顺序——置出完成后才能启动置入，整体周期更长（预计18-24个月，预估）

  • 天保基建置出地产后短期内仅有大飞机营收，可能触发ST预警

  • 星恒股东（尤其是PE）是否愿意等待两步走的时间周期存疑

两种路径对比：

4.4 不同收购比例的选择逻辑

无论采用哪种结构模式和实施路径，天保基建收购星恒的比例需要综合权衡：

收购比例 对价（中枢30亿，预估） 控制程度 财务影响

各比例的核心考量：

30%+控股权：适用于天保资金约束较紧（路径二总计约8-11亿）、部分投资人愿意换股保留的情景。关键问题是约70%的少数股东权益对天保基建的归母净利润稀释较大。

51%+绝对控制：多数场景下的最优平衡点——控制力充分、资金需求可控、为战略国资/管理层保留少量股权维持其积极性。

被动走向高比例收购意味着天保在谈判中拥有极强的定价权——投资人是「求着退出」而非「选择退出」。折扣空间可能从7折（路径一A组）降至4-5折（IPO失败后B组急迫）。

  1. 分步收购——先以4-5折收购B组急迫，利用前序价格锚定效应压低后序价格

  2. 结构化退出菜单——为不同投资人提供差异化选项：现金退出（折扣大）、换股（天保基建A股，估值公允）、可交换债（保底+上行期权）

  3. 利用国资投资人的合规诉求——深创投/安徽高新投更可能接受换股，可减轻现金压力

4.8 底线处置方案

若最坏情况发生——IPO失败后大量投资人起诉，股权冻结累积超过10%（CSRC并购重组审核的红线）：

  1. 分步装入：仅收购无争议股权（≥51%）——参照华夏幸福（600340）2018年案例

  2. 司法清理先行：以第一大股东身份设立争议股权收购基金（3-5亿，预估），推动一揽子和解

  3. 最后救济：仅收购经营性资产（而非股权）——代价是失去「星恒」品牌和法人主体连续性

五、委托加工天津平台——工信部产能新规的替代方案（简要分析）

5.1 方案定位

在工信部新建产能审批存在不确定性的背景下，委托加工模式（利用星恒现有产能，不涉及新建）成为实现天津产值落地的过渡性替代方案。本章仅做简要结论性分析——待星恒赴工信部确认具体政策后，再对该方案进行展开。

5.2 方案要点

在天津保税区设立运营实体（天保控股51%+星恒49%），以委托加工模式利用星恒现有产能（滁州/苏州/青川），产品面向北方市场和保税区出口。天津平台不新建产线，核心合规依据为：产能审批新规仅约束新建产能的行政审批，不约束存量产能的委托加工安排。

5.3 产能可用性结论

经核实星恒产线实际状态（FY2025实测数据），关键结论如下：

  • 星恒名义产能9.16GWh，但产线间存在严重的产品代际断层

  • 核心产线B2（15119/17119主流圆柱规格）利用率101-103%，已满产超负荷——无闲置产能可用于委托加工

  • B1/B3-1产线（合计2.8GWh）名义闲置，但原因是产品规格已被市场淘汰（1833格式/金砖72Ah），改造成本高、周期长

  • 仅B3-2（约0.86GWh，利用率65.2%）在爬坡期有一定剩余空间（约0.3GWh），但正在被自有订单填充

  • 委托加工的可行性取决于产线改造投入或B3-2爬坡剩余产能的可用性——不是「零CapEx快速启动」

详细产能数据和产线状态见主报告2.3.8节及附件I产线明细。

5.4 核心风险

  • 委托加工的法律约束力无法替代实际控制力——若天保仅为少数股东，星恒可随时以「产能紧张」为由压低天津平台订单优先级

  • 委托加工是「控制权的放大器」而非「替代品」——控制权到位后委托加工才能发挥实效

  • 品牌降级、利润率稀释、客户关系脱媒等长期隐患需通过保护条款设计加以约束

七、谈判策略——下周二会议议案要点

7.1 会议定位与核心信息

会议定位：天保控股与星恒管理层就收购方案进行新一轮商讨。目标：

  • 向管理层呈现天保的完整方案逻辑，强调利益共同体定位

  • 就附条件协议框架进行初步讨论

  • 就委托加工天津平台的设计征询管理层意见

  • 明确后续推进时间表

核心信息（天保向管理层的表述要点）：

  1. 天保不是「夺权者」——路径一期间（5年），管理层保留经营控制权和表决权，天保仅作为少数股东

  2. 天保是「解套者」——只有天保出资清理启源纳川僵局，IPO前置条件才能满足

  3. 附条件协议是「保险」，不是「陷阱」——IPO成功则协议终止，管理层毫发无损

  4. 委托加工是「增量」，不是「替代」——协议将明确优先保障星恒自有订单产能

  5. IPO失败不是天保希望看到的——但作为负责任的投资者，必须有备用方案

7.2 不可妥协的底线

  1. 附条件协议必须签署（公证处托管）——这是整个双层架构的核心支柱

  2. A组定价不超过7折（12.1元/股）

  3. 天保控股对天保基建持股≥34%

7.3 委托加工议题的谈判建议

在下周二会议中，坦诚讨论委托加工的产能可行性和长期隐患，通过设计保护条款（产能优先保障/定价定期重议/品牌区域划分/超额利润分成）来主动化解管理层顾虑。将委托加工定位为过渡性安排——待工信部政策确认后，再行展开详细方案。

8.3 关于退出价值的说明

星恒计划以港股为上市目的地。当前港股市场对小市值制造业公司——尤其是分红不突出、处于竞争性行业的中等规模企业——估值环境存在显著不确定性。考虑两三年后港股的市场环境更难以预测，即使IPO成功，退出估值也可能不理想，存在较大可能的本金损失风险。

准确的退出价值分析、投资回报测算和盈亏平衡分析，需参见独立的投资回报测算报告。本章仅提供资金安排的综合概览，不包含退出回报的定量估算。
