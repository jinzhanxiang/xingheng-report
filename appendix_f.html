附件F：天保控股收购星恒电源——自身立场衡量与谈判议案

一、天保控股的战略出发点与核心约束

1.1 战略动机

天保控股收购星恒电源的战略逻辑基于三重考量：

产业转型需要：天保基建（000965.SZ）主营房地产开发，ROE仅0.35%，经营现金流为负，亟需注入新兴产业资产。星恒电源作为两轮车锂电头部企业（72.5%市占率），是理想的转型标的。战略叙事为「大飞机+新能源双轮驱动」——保留大飞机板块的估值锚作用，以新能源业务重构公司基本面。

区域经济协同：星恒若在天津保税区设立运营实体，可贡献工业产值、税收和就业。委托加工模式能绕过工信部新建产能审批门槛（详见1.3节），实现产值落地天津。

国有资产保值增值：以合理价格获取星恒股权，打通IPO退出路径或A股注入通道，实现国有资本增值退出。

1.2 天保控股的核心约束

以下约束条件决定了收购方案的可行边界：

（1）资金成本约束

天保控股作为天津保税区国有资产运营平台，任何资本注入均需考虑资金成本：

资金来源 成本基准 年化成本率
天保控股自有资金注资 国有资本保值增值率要求 ≥6%（[K] 地方SOE国资委考核一般基准，具体以保税区国资委实际考核要求为准）
银行并购贷款 LPR+信用利差 4.0-5.0%（[K] 2026年商业银行并购贷款市场利率区间）
天保基建自有资金 上市公司WACC 约5-6%（[K] 地产/园区类A股公司WACC一般区间，非天保基建精确计算值）
天保控股发债融资 AA+城投债收益率 3.5-4.5%（[B] 中国债券信息网城投债收益率曲线 ★★★★☆）

含义：天保控股每向天保基建注资1亿元，年度机会成本约600万元（按6%国有资本回报率参考值计算，实际考核值需向保税区国资委确认）。若注资5亿，年度机会成本约3,000万元——需在项目回报测算中计入。建议在下周二会议前向保税区国资委确认天保控股层面对外投资的回报率考核要求。

（2）天保基建控制权红线（34%）

天保控股持有天保基建51.45%（5.71亿股/11.10亿股）。34%是《公司法》第216条的特殊决议否决权红线（一票否决修改章程/增减资/合并分立）。发行股份收购星恒将稀释天保控股持股——需确保稀释后≥34%。

最大稀释空间：5.71/(11.10+X) ≥ 0.34 → X ≤ 5.69亿股。按发行价≥每股净资产4.91元计算，最大纯发股收购规模约27.96亿——远超本方案资金需求（8.6-11亿）。

（3）天保基建破净限制

天保基建当前PB≈0.67x（股价3.33元 vs BPS 4.91元）。定增发行价≥每股净资产（国资监管要求），导致定增价比市价高47%——对星恒PE股东（认购方）意味着接受远高于市场价格的定价，接受度存疑。这一约束使得现金收购成为首期交易的首选支付方式。

（4）审批链条

  • 天保控股对外投资≥5亿：需保税区国资委审批，可能需报天津市国资委备案

  • 天保基建重大资产重组：需深交所审核+CSRC注册

  • 涉及国有资产转让：适用32号令（《企业国有资产交易监督管理办法》）

（5）房地产跨界标签

CSRC对房地产上市公司跨界并购窗口指导趋严。需在收购推进前论证「主业转型」（大飞机+新能源双轮驱动），建议同步启动地产板块剥离的可行性论证。

1.3 工信部产能政策冲击（概述）

根据行业初步信息，工信部正在推进电池产能审批机制改革，核心变化：

  • 审批权限上收至中央，实行指标制+分类管控

  • 存量已批项目不受影响

  • 新建产能硬性门槛大幅抬高（高能量密度/长循环/碳足迹）

  • 新规真正收紧窗口从2027年下半年开始

对天保收购方案的核心影响：原计划中「10GWh天津产能落地」在新规下面临行政审批障碍，无法作为当前阶段的确定要素。委托加工模式（利用星恒现有产能，不涉及新建）成为实现天津产值落地的务实替代方案。

重要提示：上述信息为行业初步判断，尚未经工信部正式确认。建议在下周二会议中与星恒管理层商议：由星恒赴工信部确认具体政策内容和时间表。不排除产能置换等替代路径。

三、天保控股的最优收购方案——「轻介入+附条件升级」

3.1 方案总览

天保不在一开始追求控股，而是采用双层架构：

路径 触发条件 天保持股 资金需求 核心特征
路径一（轻介入） 当前即启动 13.73% 5亿 少数股东+附条件协议推动IPO
路径二（控股并表） IPO失败后自动触发 30%+（含管理层表决权） 8-11亿（含B组增持） 控股股东+替代路径（装入天保基建A股）

核心机制：附上市条件协议连接两条路径。IPO成功→天保少数股权退出，管理层保留一切；IPO失败→管理层表决权自动委托天保→天保从13.73%升级为19.99%第一大股东→低价增持B组至30%+→控股并表→启动替代路径。

4.3 路径B（天保能源）基本排除的详细理由

天保能源（01671.HK）市值仅约1.36亿HKD，而星恒营收约36亿RMB（约39亿HKD）。任何可感知的星恒股权注入（哪怕仅5%）即触发HKEX Rule 14.06B反向收购（RTO=新上市申请标准）。RTO须满足Rule 8.05(c)（3个财年在大致相同管理层及拥有权下经营），等待期与独立IPO相同，但增加了关连交易+极端VSA+RTO三层复杂度。

4.4 底线处置方案

若最坏情况发生——IPO失败后大量投资人起诉，股权冻结累积超过10%（CSRC并购重组审核的红线）：

  1. 分步装入：仅收购无争议股权（≥51%）——参照华夏幸福（600340）2018年案例

  2. 司法清理先行：以第一大股东身份设立「争议股权收购基金」（3-5亿），推动一揽子和解

  3. 最后救济：仅收购经营性资产（而非股权）——代价是失去「星恒」品牌和法人主体连续性

五、委托加工天津平台——审慎评估（含产能可行性核实）

5.1 方案概述

在天津保税区设立运营实体（「天保星恒新能源有限公司」，天保控股51%+星恒49%），以委托加工模式利用星恒现有产能（滁州/苏州/青川），产品面向北方市场和保税区出口。天津平台不新建产线，核心合规依据为：产能审批新规仅约束新建产能的行政审批，不约束存量产能的委托加工关系。

委托加工的法律合规框架与产能审批兼容性（点击展开）

法律依据：《民法典》第770条（承揽合同）；《产品质量法》第26-27条（委托方承担产品质量责任）；《环境保护法》（生产环节环保责任由承揽方承担）。

与产能审批新规的兼容性：新规针对新增产能的行政审批，委托加工不新建产线、不申请新增产能指标、不新增行业总产能——新规的字面含义和立法目的均不覆盖委托加工安排。

需关注的合规边界：若星恒为满足天津平台订单而扩建产线→触发产能审批新规；若天津平台提供资金/设备用于扩产→可能被视为变相新增产能。

5.2 核心审视——星恒产能结构核实与委托加工可行性

此前分析将星恒产能利用率简化为「60-70%」，隐含假设为「有30-40%闲置产能可用于委托加工」。经核实历史分析记录（附件I天津10GWh产能落地纵横分析），这一假设不成立。

5.2.1 星恒产线的真实状态（FY2025实测数据）

星恒现有名义产能9.16GWh（苏州+滁州），但产线之间存在严重的产品代际断层——产能不是「量」的过剩，而是「质」的结构性错配：

B3-1 金砖72（72Ah） 0.8GWh 0% 已淘汰——客户转向100Ah+，72Ah产品无市场需求 不可用——产品已被市场淘汰，产线需要改造才能生产当前规格产品

  • 名义利用率65.8%（行业口径）掩盖了结构真相

  • B3-1（0.8GWh）+ B1（2.0GWh）= 2.8GWh「闲置」产能，但闲置原因是产品已被市场淘汰，不是需求不足

  • 核心产线B2（15119/17119规格）利用率101-103%，已经超负荷

  • B3-2（65.2%）在爬坡期，但随着订单释放，剩余空间正在被填充

5.2.2 委托加工产能可行性的重新判断

前提错误：此前假设「闲置产能30-40%可被委托加工调用」。实际是：

  • 可用的B2线（主流产品）已满产超负荷 → 委托加工订单将与星恒自有订单直接竞争产线时间

  • 闲置的B3-1/B1线只能生产过时产品 → 产线改造需要资本投入（估计数千万至亿元级）+ 6-12个月改造周期 → 不是「快速启动」方案

委托加工的真实可行性取决于以下任一条件：

  1. 星恒对B3-1/B1产线进行技术改造（投入CapEx+时间），使其能生产当前规格产品

  2. 委托加工使用B3-2爬坡期的剩余产能（约35%，但正在被填充）

  3. 委托加工产品恰好是金砖72/1833等过时规格（无市场需求，不成立）

最可能的情景：委托加工初期可使用B3-2剩余产能（约0.3GWh），但长期需依赖产线改造（B3-1/B1）或B3-2扩产，两者均涉及CapEx投入和审批风险。委托加工的「零CapEx快速启动」优势被高估——轻资产是指天津平台不需投资，但星恒端的产能释放可能需要投资。

5.2.3 对星恒的真实利弊（修正后）

短期利益（修正）：

  • 有限消化B3-2爬坡期剩余产能（约0.3GWh，正边际贡献）

  • 不新增资本开支（但前提是B3-2剩余产能足够）

  • 天津平台承担品牌和市场风险

长期隐患（维持原分析，新增产能竞争风险）：

  • 若B2线持续满产，委托加工订单与星恒自有订单竞争产线时间 → 可能被迫在「保自有客户」和「保天津平台」之间做出选择

  • 品牌降级风险、利润率稀释、客户关系脱媒（同此前分析）

  • 若星恒需改造B3-1/B1产线以支持委托加工 → 新增CapEx由谁承担成为新的谈判议题

对天保的战略价值（修正）：

  • 合规绕过产能审批门槛——仍成立（不涉及新建产能）

  • 快速启动——从6-9个月修正为9-18个月（考虑到产线可行性验证+可能的改造周期）

  • 产值归属天津——仍成立

  • 轻资产——天津平台端成立，但星恒端的产能实现可能需要CapEx

5.3 委托加工实效依赖控制权

委托加工协议的法律约束力无法替代实际控制力。若天保仅为13.73%少数股东，星恒可随时以「产能紧张」（B2线已102%满产是客观事实）为由压低天津平台订单优先级。委托加工不是「控制权的替代品」，而是「控制权的放大器」。 路径一（13.73%+控制董事会→并表）是委托加工天津平台发挥实效的必要前提——只有天保在董事会层面能影响产能分配决策，天津平台的订单才能获得优先保障。

八、谈判策略——下周二会议议案要点

8.1 会议定位与核心信息

会议定位：天保控股与星恒管理层的首次正式收购方案讨论。目标：

  • 向管理层呈现天保的完整方案逻辑，强调「利益共同体」定位

  • 明确后续推进时间表

核心信息（天保向管理层的表述要点）：

  1. 天保不是「夺权者」——路径一期间（5年），管理层保留经营控制权和表决权，天保仅作为少数股东

  2. 天保是「解套者」——只有天保出资清理启源纳川僵局，IPO前置条件才能满足

  3. 附条件协议是「保险」，不是「陷阱」——IPO成功则协议终止，管理层毫发无损

  4. 委托加工是「增量」，不是「替代」——协议将明确优先保障星恒自有订单产能

  5. IPO失败不是天保希望看到的——但作为负责任的投资者，必须有备用方案

8.2 不可妥协的底线

  1. 附条件协议必须签署（公证处托管）——这是整个双层架构的核心支柱

  4. 天保控股对天保基建持股≥34%

  • 将委托加工定位为「过渡性安排」——若星恒自有订单增长到填满产能，可调减委托加工量或提高加工费率
