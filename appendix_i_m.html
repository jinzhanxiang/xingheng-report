零、逻辑起点：星恒电源在产业链中的锚定位置

0.1 星恒是什么？

星恒电源是两轮车/低速电动车用锂电电芯制造商，技术路线为LMFP（磷酸锰铁锂）固相法。 2025年ROE约1.4%，净利率约1.24%，天保控股拟以PS 1.0x入股（合理PS为0.23-0.51x）。

前提（已验证）: 星恒电源不是好的投资标的。详见已有尽调报告。

核心问题: 星恒不赚钱，那钱被谁赚走了？是上游供应商，还是下游客户？

  • 推论1: 若上游供应商毛利率显著高于星恒 → 利润在上游沉淀 → 上游值得投资

  • 推论2: 若下游客户ROE显著高于星恒 → 利润被下游捕获 → 下游值得投资

  • 推论3: 若全链都不赚钱 → 该细分产业链条整体回避

三、反向辩论：对核心结论的系统性质疑

🟡 CAUTION

🟡 本章刻意站在「上游优于下游」结论的对立面，以最严格的标准审视该结论可能存在的逻辑漏洞、混淆变量和未证前提。这是纵横分析法的内在要求——任何结论必须经受反向辩论的考验后才能成立。

3.1 质疑一：上游利润暴增是否仅是大宗商品价格周期？

前提2: 2025年碳酸锂产量54.8万吨，同比+22.8%
前提3: 2024年多矿企计提大额资产减值（周期底部特征），2025年减值大幅减少

──────────────────────────────────────────────────────────
分解结论:
  - 价格贡献: ~0%（均价持平）
  - 销量贡献: ~23%（产量增长）
  - 成本+减值改善: ~77%（减值是2024年利润被压至近零的主因）
  
  2025年上游利润增长主要是: (a)量增 + (b)减值回补 + (c)成本结构优化
  而非碳酸锂价格上涨。
  
  这意味着: 2025年的利润修复有结构性成分（低成本产能替代高成本产能），
  而非纯粹的「涨价周期」。
置信度: ⭐⭐⭐⭐（减值数据为全行业汇总，未逐家验证）

矛盾解决的核心逻辑:

  利润迁移不是均匀地流向「所有上游环节」，
  而是精确地流向「具有定价权的上游环节」。

  定价权来源:
  1. 资源稀缺性 → 盐湖/锂矿/钴矿 → ✅ 有定价权

  3. 低集中度(CR3<30%) → 正极材料 → ❌ 无定价权，被电芯厂宰割

  但CR3不到30%，387万吨产量严重过剩，

  - 迁移到「强势上游」（资源+高集中度主材）
  - 放弃「弱势上游」（低集中度主材+重资产环节）
  
  德方纳米亏损不构成对利润迁移论的反驳，

核心事实：前驱体仅占电芯总成本的5-8%。这意味着即使前驱体环节发生革命性变化，对电芯总成本的影响也极为有限。

6.3.2 固相法 vs 液相法：技术优势的量化边界

维度 固相法（星恒路线） 液相法（德方纳米路线） 固相法优势
工艺复杂度 低（固相混合+烧结） 高（溶液共沉淀+烧结） ✅ 设备投资低30-40%
碳酸锂要求 必须用电池级Li₂CO₃ 可用工业级Li₂CO₃ ❌ 原料成本反而更高
前驱体成本 约2.7-3.0万/吨 约2.5-2.8万/吨 ❌ 实际成本略高5-10%
产品一致性 中等（固相混合均匀性受限） 好（溶液级混合） ❌ 批次稳定性不如液相法

数据来源: 星恒商业尽调报告 + 德方纳米招股书 + 高工锂电技术路线对比 | 置信度: ⭐⭐⭐⭐

关键发现：固相法的「低成本」优势是一个需要精确辨析的命题。

固相法的真实成本画像:

  优势侧:
  ✅ 设备投资低30-40% → 折旧低 → 制造成本每吨低约500-800元
  ✅ 工艺简单 → 良率可达95%+ → 废料损耗低
  ✅ 扩产快 → 适合两轮车市场快速起量

  劣势侧:
  ❌ 必须用电池级碳酸锂（比工业级贵约5-10%）→ 原料成本每吨高约1000-1500元
  ❌ 产品一致性不如液相法 → 不适合动力电池客户
  ❌ 综合来看，固相法前驱体总成本与液相法基本持平甚至略高

  → 固相法的优势不在「成本更低」，而在「设备投资更低+扩产更快+适配两轮车」。
    这是「工艺适配优势」而非「成本碾压优势」。
置信度: ⭐⭐⭐⭐

6.3.3 量化测算：固相法成功对星恒利润的影响

设星恒当前状况：

  • 年营收36.36亿（FY2025审计数据）

  • 净利率1.3%（净利约4700万）

  • 毛利率16.5%

  • 前驱体自供比例约X%（星恒固相法产线主要用于自供）

情景一：固相法前驱体成本降低30%（乐观假设）

前提: 前驱体占电芯成本5-8%，取中值6.5%
      前驱体成本降低30% → 电芯总成本降低 6.5% × 30% = 1.95%

营收36.36亿，成本率83.5%（毛利率16.5%）
  → 营业成本 = 30.36亿
  → 前驱体成本 = 30.36 × 6.5% = 1.97亿
  → 前驱体成本节省 = 1.97 × 30% = 0.59亿

假设节省全部转化为利润（不考虑竞争降价）:
  → 新增净利 = 0.59亿 × (1-15%所得税) = 0.50亿
  → 净利从4700万提升至9700万
  → 净利率从1.3%提升至2.67%
  → ROE从1.4%提升至2.87%（权益27.95亿）

情景二：固相法前驱体成本降低10%（中性假设）

前驱体成本节省 = 1.97 × 10% = 0.197亿
新增净利 = 0.197 × 0.85 = 0.17亿
净利从4700万提升至6400万
净利率从1.3%提升至1.76%
ROE从1.4%提升至1.91%

情景三：固相法前驱体成本降低50%（极度乐观，技术上几乎不可能）

前驱体成本节省 = 1.97 × 50% = 0.99亿
新增净利 = 0.99 × 0.85 = 0.84亿
净利从4700万提升至1.31亿
净利率从1.3%提升至3.60%
ROE从1.4%提升至3.91%

计算基于: 星恒FY2025审计数据（中天运会计师事务所） | 置信度: ⭐⭐⭐⭐

核心结论:

→ 推论23: 即使固相法取得「极度成功」（前驱体成本降50%），星恒ROE也只能从1.4%提升至约3.9%。
  在更现实的中性假设下（成本降10%），ROE仅提升至1.9%。
  这一改善幅度无法改变星恒「不具投资价值」的基本判断。

原因很简单:
  1. 前驱体只占电芯成本5-8%，天花板极低
  2. 固相法的成本优势被「必须使用电池级碳酸锂」这一硬约束部分抵消
  3. 星恒面临的是结构性问题（下游议价权+固定成本负担+两轮车TAM有限），
     非单一技术进步所能解决

类比: 这就像一家餐厅的盐成本降低了30%——盐只占菜品成本不到1%，
      即使免费获得盐，也不会让餐厅的利润率发生质变。
置信度: ⭐⭐⭐⭐⭐
